# 🃏 牌友 - 快速测试指南

## ✅ 当前功能

### 炸金花游戏（儿童简化版）

**游戏流程：**
1. 打开小程序 → 点击"🎴 炸金花"
2. 系统自动发牌（有发牌动画）
3. 查看自己的手牌和牌型（带 emoji 提示）
4. 点击"🎲 开牌比大小"
5. 显示双方牌面和牌型
6. 自动判定胜负，显示结果弹窗
7. 赢了有庆祝特效 🎉
8. 点击"🃏 再来一局"继续

**牌型说明（从大到小）：**
- 🎉 豹子 - 三张一样的牌
- ✨ 顺金 - 同花顺
- 🌟 金花 - 同花
- 📶 顺子 - 连续三张牌
- 👫 对子 - 两张一样的牌
- 🂡 单张 - 比最大牌

**积分规则：**
- 赢：+20 分
- 平：+10 分（鼓励小朋友）
- 输：-10 分（积分不会变负数）

## 📱 在微信开发者工具中测试

1. 打开微信开发者工具
2. 导入 `card-buddy` 项目
3. 如果提示 AppID，可以：
   - 使用测试号
   - 或填入自己的小程序 AppID
4. 点击"编译"按钮
5. 在模拟器中点击"炸金花"开始测试
