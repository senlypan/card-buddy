## 💰 摸金模式玩法

### 游戏流程

```
1️⃣ 选择难度入场
   ↓
2️⃣ 探索建筑
   ├─ 搜索物资箱
   ├─ 击败 AI 守卫
   ├─ 开启保险箱
   └─ 寻找 BOSS 房间
   ↓
3️⃣ 击败 BOSS（可选）
   ↓
4️⃣ 找到撤离点
   ↓
5️⃣ 成功撤离 → 保存战利品
```

### 失败条件

- ❌ 血量归零
- ❌ 时间到（5 分钟）

### 成功条件

- ✅ 到达撤离点并撤离

## 🎯 经济系统

### 货币类型

- **💎 积分**：获胜获得，解锁模式
- **🪙 金币**：摸金获得，购买道具
- **🔧 机甲碎片**：合成机甲

### 获取方式

| 行为 | 奖励 |
|------|------|
| 经典模式胜利 | +10~30 积分 |
| 摸金成功撤离 | +50~100 金币 |
| 击败敌人 | +10~40 金币 |
| 击败 BOSS | +200~600 金币 |
| 出售宝物 | 10~1000 金币 |
