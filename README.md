# 🃏 牌友 - 儿童陪玩扑克游戏

一个专为儿童设计的微信小程序扑克游戏，陪伴孩子快乐成长。

## 🎯 项目定位

- **目标用户：** 6-10 岁儿童（小学低年级）
- **核心价值：** 情感陪伴 + 适度娱乐
- **人格设定：** "牌牌" - 活泼友好的 AI 玩伴

## ✨ 已实现功能

### 首页
- 🃏 牌友打招呼互动
- 📊 积分实时显示
- 🎮 游戏选择入口
- 💡 游戏时间提醒

### 炸金花（完整版）

- ℹ️ 关于页面

- **大按钮设计** - 方便小手点击

- 渐变背景色

## 🚀 本地调试

### 环境准备
1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号（个人类型即可）

### 运行步骤
```bash
# 1. 克隆项目
git clone <repository-url>
cd card-buddy

# 2. 打开微信开发者工具
# 选择"导入项目"，选择 card-buddy 目录

# 3. 配置 AppID
# 在 project.config.json 中填入你的小程序 AppID
# 或使用"测试号"模式

# 4. 编译运行
# 点击"编译"按钮即可在模拟器中预览
```

### 真机调试
1. 微信开发者工具 → 预览
2. 扫描二维码在手机上测试

## 🎮 游戏规则（炸金花）

### 牌型大小（从大到小）
1. **豹子** - 三张相同点数
2. **顺金** - 同花顺
3. **金花** - 同花
4. **顺子** - 顺子
5. **对子** - 两张相同
6. **单张** - 比最大牌

### 积分规则
- 赢一局：+20 分
- 平局：+10 分（鼓励孩子）
- 输一局：-10 分
- 弃牌：-10 分
