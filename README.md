# 🃏 牌友 - 儿童陪玩扑克游戏

一个专为儿童设计的微信小程序扑克游戏，陪伴孩子快乐成长。

## 🎯 项目定位

- **目标用户：** 6-10 岁儿童（小学低年级）
- **核心价值：** 情感陪伴 + 适度娱乐 + 思维锻炼
- **人格设定：** "牌牌" - 活泼友好的 AI 玩伴

## 📖 游戏规则

### 炸金花

**牌型大小（从大到小）:**
1. 🎉 豹子 - 三张相同（AAA 最大）
2. ✨ 顺金 - 同花顺
3. 🌟 金花 - 同花
4. 📶 顺子 - 连续三张
5. 👫 对子 - 两张相同
6. 🂡 单张 - 比最大牌

**积分规则:**
- 赢一局：+20 分 🎉
- 平局：+10 分 ✨
- 输一局：-10 分 💪（不会变负）

**散牌点数规则:**
- A = 1 点，2-10 = 牌面数字，JQK = 10 点
- 三张牌总点数取个位（0-9 点）
- 例如：K+Q+J = 30 点 = 0 点，9+8+7 = 24 点 = 4 点

### 斗地主

**牌型:**
- 单张、对子、三张、三带一、三带二
- 顺子（5 张以上连续）、连对（3 对以上连续）
- 飞机（连续三张）、炸弹（四张相同）、王炸（大王 + 小王）

**牌型大小:**
- 大王 > 小王 > 2 > A > K > Q > J > 10 > ... > 3

**游戏流程:**
1. 选择地主/农民
2. 发牌（地主 20 张，农民 17 张）
3. 地主先出牌
4. 轮流出牌，管上或不要
5. 先出完牌者获胜
